# 使用说明

## 1. 开始前

因子罗盘支持 Windows 10/11 x64。下载 Release 中的便携 ZIP，完整解压后运行 `Sigil-Compass.exe`。

程序没有商业代码签名。SmartScreen 出现时，先确认下载地址是：

```text
https://github.com/SalmonC/gbfr-sigil-compass/releases
```

应用不会修改存档，但仍建议在首次使用前手动备份。

## 2. 读取存档

按 `Win + R`，输入：

```text
%LOCALAPPDATA%\GBFR\Saved\SaveGames\
```

通常选择 `SaveData1.dat`。有多个文件时，优先选择修改时间最新的 `SaveData*.dat`。

点击“读取存档”后，工具会：

1. 把源文件复制到应用私有临时目录；
2. 校验并只读解析临时副本；
3. 保存解析后的因子清单；
4. 删除临时副本。

再次打开程序时会显示上次成功读取的库存，但不会自动访问原存档。游戏内获得、强化或分解因子后，请重新读取存档。

## 3. 设置目标

先点击右侧一个目标区域，再点击左侧技能池中的技能。

- **必须满足**：缺少任意一项时，不显示方案；同一技能可以重复添加。
- **基础属性主词条**：要求技能出现在因子的第一个词条。允许替代时，可以设置可接受技能及顺序。
- **攻击类主词条 / 防御类主词条**：指定相应分类的主词条数量和内容。
- **可选目标**：满足高优先级要求后尽量凑齐，顺序越靠前越重要。
- **不能出现的技能**：任意所选因子带有该技能，整套方案直接排除。
- **尽量避开的技能**：方案仍会显示，但排名降低，并在结果中标出。

点击目标里的技能可以删除；每个区域右上角都有单独清空按钮。当前不可添加的技能会变灰，鼠标移上去可查看原因。

## 4. 查看计算结果

点击“开始分析”后，页面会自动滚动到结果区。

- 最多显示排名前 10 的逻辑方案。
- 标签页用于切换方案；出现替代主词条或“尽量避开”技能时会显示警告。
- 每枚因子卡片会列出主词条、副词条和因子等级；不在目标中的附带词条会弱化显示。
- 因子列表下方按技能汇总等级，例如两条 15 级追击显示为“追击 Lv 30”，不显示因子等级总和。
- 主、副词条都可从真实未占用库存中选择替代，也可以更换、删除或添加整枚因子。调整版保存在原排名结果下，不改变排名。
- 技能组合相同但等级不同的因子属于同一逻辑方案；实际选取时优先使用高等级实例。
- `A&B` 与 `B&A` 的主词条不同，因此算作不同方案。
- 目标外副词条只有位于相同主词条位置时才会合并；位置不同仍保留为独立方案，以保留不同的替换范围。

达到计算资源上限时，工具会停止并提示，不会把未完成结果当作最优方案。可尝试减少可选目标，或增加“不能出现”的技能。

## 5. 手动调整结果

主、副词条选择框对应的是整枚真实因子，不是凭空修改技能：

- 改副词条时固定主词条，只显示主词条相同的可用因子；
- 改主词条时固定副词条，只显示副词条相同的可用因子；
- 选项中的等级和库存位置用于区分内容相同的不同实例。

卡片右上角可以更换或删除整枚因子；少于 12 枚时可以继续添加。候选不包含当前结果的其他因子，也不包含其他已确认配装占用的因子。

编辑后会立即更新目标缺口、警告和技能等级汇总。调整版不会重新参加前十名排序，可以随时恢复为原计算结果。缺少必须满足的技能或带有“不能出现”的技能时，确认按钮不可用。

## 6. 保存、分享和重新计算

方案名称、目标和最近一次结果会自动保存在本机。可以：

- 复制为新方案；
- 在顶部下拉框切换方案；
- 导出带名称和顺序的分享字符串；
- 导入别人提供的分享字符串；
- 库存更新后重新计算。

分享字符串不包含存档路径、库存或玩家身份。

原始计算结果和手动调整版会跟随当前方案缓存。目标、库存或占用发生变化后，无法继续使用的旧结果会被清理。

## 7. 多角色配装

查看某套结果时，可以点击“确认配装”。确认后，占用的是具体因子实例，而不是整个技能组合；其他同词条因子仍可供别的角色使用。

手动调整过的结果按调整后的具体实例确认。“已确认配装”页面可以查看和取消现有占用。如果重新读取存档后某枚因子消失，工具会标出缺失，不会静默换成另一枚。

## 8. 查看库存

“持有因子”页面支持：

- 搜索主副词条；
- 按技能分类、可用状态和游戏内装备状态筛选；
- 按槽位、等级、主词条或副词条排序；
- 查看已被哪套确认配装占用。

库存页面显示的是上次手动读取后的快照，不代表游戏文件的实时状态。
