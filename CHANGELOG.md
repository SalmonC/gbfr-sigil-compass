# 更新记录

## 未发布

- 项目原创代码和文档改用 MIT License；游戏内容及第三方素材不在该授权范围内。

## v0.2.1

- 单次计算时间上限可在界面设置为 5–600 秒，默认 30 秒并记住上次设置。
- 分开提示“计算超时”和“组合状态超过安全容量”。
- 顶部增加 GitHub 项目入口。

## v0.2.0

- 在读取按钮旁补充默认存档目录和文件选择说明。
- 精确求解时间预算提高到 30 秒，外层 Worker 在 35 秒回收。
- 放宽求解状态容量，仍然拒绝返回不完整结果。
- 修复 Windows 宽屏下技能池内容高度与外框不一致的问题。
- 增加相应回归检查。

## v0.1.0

- 首个可用版本。
- 支持只读存档导入、目标编辑、精确 Top-10 配装、方案缓存与分享。
- 支持库存查看、确认配装和多角色具体因子占用。
