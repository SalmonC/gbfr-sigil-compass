# 隐私说明

因子罗盘是离线、本地优先工具，不提供账号系统、遥测或云同步。

## 会读取的内容

- 用户手动选择的 GBFR `SaveData*.dat`；
- 应用自己的本地配置、库存快照和方案缓存。

读取存档时，程序先复制到应用私有临时目录，再由只读解析进程处理。无论成功或失败，临时目录都会进入清理流程。程序不写回源存档。

## 会保存的内容

- 最近一次成功读取的源路径；
- 解析后的双词条因子清单和读取时间；
- 命名方案、目标、分享配置和最近一次计算结果；
- 已确认配装使用的具体因子实例；
- Electron/WebView 运行所需的少量本地数据。

程序不保存源存档副本，也不会自动重新打开上次的存档。

## 网络

正式应用不需要联网，不上传存档、库存、目标、结果或诊断信息。下载新版本、查看文档和向 GitHub 提交反馈由用户自行完成。

## 清除数据

关闭应用后，删除以下目录即可清空应用数据：

```text
%APPDATA%\Sigil Compass\
```

这不会影响游戏存档。不同构建可能使用略有不同的目录名称，请在删除前按内容和修改时间确认。

## 反馈注意事项

不要在公开 Issue、截图或日志中附带完整存档、Windows 用户名、绝对路径或其他个人信息。需要私下报告安全问题时，请使用仓库 Security 页面。
