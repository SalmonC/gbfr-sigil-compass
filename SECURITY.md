# 安全说明

## 支持范围

当前只维护 GitHub Releases 中最新的 Windows 版本。旧版问题可能需要先在最新版复现。

## 报告安全问题

请不要为以下内容创建公开 Issue：

- 存档或本地文件被意外读取、修改或泄露；
- 路径授权、IPC、临时文件或缓存存在越权问题；
- 可执行文件加载不受信任内容；
- 其他可能影响用户数据或系统安全的问题。

优先使用 GitHub 仓库 Security 页面的私密漏洞报告功能。如果该入口暂不可用，请在 Issue 中只说明“需要私下报告安全问题”，不要附带复现细节、存档或敏感路径。

报告建议包含受影响版本、Windows 版本、复现步骤、影响范围和可行的缓解方式。维护者确认前，请勿公开完整利用方法。

## 存档样本

不要直接发送真实存档。若问题确实只能通过样本复现，请先等待维护者提供私下传递和删除方案。
